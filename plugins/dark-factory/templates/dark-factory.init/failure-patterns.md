# Dark Factory Failure Patterns

Known failure modes and recovery strategies. Pipeline Doctor appends entries here after each diagnosis.

## Format

```markdown
### [DATE] [LAYER] [ERROR-TYPE]: Brief description
- **Root Cause**: What actually went wrong
- **Fix Applied**: How it was resolved (or "ESCALATED" if fix failed)
- **Prevention**: How to avoid this in the future
- **Session**: session-id
```
